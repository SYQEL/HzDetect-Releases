# HZ Detect — Desktop Releases

Official downloads and auto-update feed for the **HZ Detect** desktop app
(macOS & Windows) — analyze the true tuning of your music library
(440 / 432 / Solfeggio) and retune it to any target frequency.

- **Download:** grab the installer for your platform from the
  [latest release](https://github.com/SYQEL/HzDetect-Releases/releases/latest)
- **Website:** https://hzdetect.com
- **Support:** support@hzdetect.com

This repository hosts release artifacts and the updater manifest only;
the application source is developed privately by SYQEL INC.
